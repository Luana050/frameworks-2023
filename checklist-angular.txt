#----------------- Rotinas do GIT
1. git status (verificar situação)
2. se vermelho -> git add .
3. se verde -> git commit -m "nome do commit"
4. se td ok! git push (atualiza no github)

5. git clone endereço.git . (comando para copiar do github para o local)
5. cd my-app
5.1 npm install (dentro da pasta do projeto para instalar node_modules)
